#### How to install

* Clone & Install

```sh
git clone ...........
cd da_design_server_학번
pip3 install -r requirements.txt
```

* Append a python path to `~/.bashrc` file.

```sh
~$ cat >> ~/.bashrc
export PYTHONPATH=$PYTHONPATH:/home/u1011/
Ctrl+d

~$ source ~/.bashrc
```
